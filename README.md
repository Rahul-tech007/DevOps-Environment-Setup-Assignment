This project documents the setup and management of secure development environments for two new developers, Sarah and Mike, under the guidance of Rahul (Senior DevOps Engineer).
The environment ensures:

System Monitoring for performance and capacity planning

User Management & Access Control for secure workspaces

Automated Backup Procedures for Apache and Nginx web servers

Task 1: System Monitoring Setup
Objective
Configure monitoring tools to track CPU, memory, processes, and disk usage for troubleshooting and capacity planning.

Task 2: User Management and Access Control
Objective
Create secure user accounts for Sarah and Mike with isolated directories and enforce password policies.

Task 3: Backup Configuration for Web Servers
Objective
Automate weekly backups for Apache (Sarah) and Nginx (Mike) servers.

Deliverables
Screenshots/Terminal Outputs: Monitoring tools, user creation, cron jobs, backup verification.

Logs: System monitoring logs, backup verification logs.

Report: Summarizing implementation steps, challenges, and solutions.

Conclusion
This setup ensures:

Secure developer environments
Continuous system monitoring
Reliable automated backups

TechCorp’s development infrastructure is now compliant with operational and security standards.
